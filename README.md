# HNG Stage 0 - Todo Card Component

This is a simple, responsive, and accessible Todo Card built as part of the HNG Frontend Wizards Stage 0 task.

##  Project Overview

The goal of this project is to build a single Todo/Task Card component that demonstrates:
- Clean UI structure
- Semantic HTML
- Accessibility best practices
- Testable elements using `data-testid`
- Basic interactivity using JavaScript

---

## Features

- Displays a task title and description
- Shows priority level (Low / Medium / High)
- Displays due date in a readable format
- Calculates and updates time remaining dynamically
- Shows task status (Pending / Done)
- Checkbox to mark task as complete
- Tags for categorization
- Edit and Delete action buttons
- Fully responsive design (mobile-first)
- Keyboard accessible elements

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
